# Worlds Reincarnate
 
Minecraft Converter & Editor!
*This is currently in early development, more editions are slated for support in the future. Have mercy on me, for I am a solo developer in my free time*
    
Take your world cross edition with this handy little tool! Designed with true behavior in mind, worlds should differ as *minimally* as possible following conversion. Releases currently built for Linux and Windows.

## Currently supported editions
* Java 
    - Pre-Classic Format (pc-132011 through c0.0.12a_03)
    - Early Classic Format (c0.0.13a-launcher through c0.0.13a_03-launcher)
* Classic Javascript
    - Raw Json Format
    - Firefox Local Storage Format
